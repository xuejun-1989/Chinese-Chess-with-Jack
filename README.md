# ChineseChessWithJack
基于 EasyX 图形库的中国象棋游戏，支持双人对战、人机对战，包含“雾刃”、“隐身”等特殊技能。

## 开发环境
- Visual Studio 2022 + EasyX (https://easyx.cn/)
- 需要安装 EasyX 图形库

## 编译运行
1. 用 VS 打开 `.sln` 文件（或新建工程导入源文件）
2. 安装 EasyX（官网下载安装包一键安装）
3. 将 `res/` 文件夹放在输出目录（或设置工作目录为项目根目录）
4. 编译运行
5. 下载src，res，slnx即可，其余不需要

## 音效 / 图片资源
所有资源放在 `res/` 子目录下：
- `images/` 包含棋子、按钮、特效图片
- `sounds/` 包含吃子、落子、将军、技能、悔棋、背景音乐等 mp3

## 游戏规则
- 标准中国象棋规则，增加两种特殊技能：雾刃（远程攻击）和隐身（车/兵瞬移）。
- 人机对战 AI 使用 Alpha-Beta 剪枝算法。
